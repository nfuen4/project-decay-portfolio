# Project Decay Portfolio Showcase

This repository contains the interactive single-page B2B case-study showcase for the **Project Decay** early-intervention health orchestration initiative.

* **Live Showcase (GitHub Pages):** [https://nfuen4.github.io/project-decay-portfolio/#visuals](https://nfuen4.github.io/project-decay-portfolio/#visuals)
